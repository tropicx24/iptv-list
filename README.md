# 📺 IPTV Otomatik Güncelleme

Bu proje IPTV playlist'inizi otomatik olarak günceller.

## 🔄 Çalışma Sistemi
- Günde 2 kez otomatik kontrol (09:00 ve 21:00)
- Çalışmayan linkler otomatik temizlenir
- Manuel çalıştırma imkanı
